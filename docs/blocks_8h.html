Tags zur Zuordnung der Überschneidungsbereiche.

Gibt an, in welche Richtung der Überschneidungsbereich gesendet wird. Sendet zum Beispiel ein Block ihren Überschneidungsbereich an den Block links von ihm, so wird der Tag T_SEND_E verwendet.
